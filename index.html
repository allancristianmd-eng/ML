Supabase (Banco de Dados)
Como obter: No painel do Supabase → seu projeto → Settings → API → copie a "Project URL" e "anon public key".
Guia de Configuração (Supabase)
1
Crie um projeto no Supabase
Acesse supabase.com → New Project. Copie a URL e a anon key.
2
Execute o Schema SQL
No Supabase → SQL Editor → cole o schema clicando em "Ver Schema SQL" acima.
3
Insira a URL e Anon Key
Cole as credenciais no card "Supabase" acima e clique em "Salvar e Testar".
4
Comece a usar
Cadastre seus produtos, adicione pedidos manualmente e gerencie seu estoque.
